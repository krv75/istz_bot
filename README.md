# istz_bot

Student Info Telegram Bot
Telegram-бот для студентов учебной группы.
Предоставляет доступ к расписанию занятий, экзаменам, дедлайнам и учебным материалам.
Администратор управляет данными напрямую через интерфейс бота.

Возможности
Для студентов
Регистрация через Telegram
Расписание занятий
Расписание звонков
Экзамены и зачёты
Дедлайны
Сроки проведения сессии
ФИО преподавателей по предметам
Учебные и методические материалы (Google Drive)
Полезные ссылки

Для администратора
Добавление данных:
сроки сессии
дедлайны
экзамены и зачёты
преподаватели
расписание занятий
Редактирование и удаление данных
Ограничение доступа по Telegram ID

Быстрый старт
Требования
Python 3.10+
PostgreSQL
Токен Telegram-бота
Google Drive Service Account (для материалов)

Запуск
python run.py

При запуске бот:
подключается к PostgreSQL
автоматически создаёт таблицы
начинает polling
